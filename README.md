# awesome-tools

## cut-pic.sh

It will cut the pic to 900\*1600, it's based on the FFmpeg.

### usage

```sh
sh cut-pic.sh
```

Then type your pic's folder

```sh
/user/XXXX/pics/
```

It will auto-cut pic and rename the pics.
